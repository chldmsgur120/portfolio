# 어린이공원 입지 분석 프로젝트  
서울 광진구 대상 어린이공원 신규 설치를 위한 입지 적정성 평가

---

## 프로젝트 개요

서울 광진구의 **낮은 어린이공원 보급률(서울 평균 대비 부족)** 과  
**유아·아동 인구 증가**, **주거 밀집**, **생활 SOC 격차** 문제를 완화하기 위해  
신규 어린이공원 설치가 필요한 지역을 데이터 기반으로 도출한 프로젝트입니다.

**QGIS 기반 공간 분석 + Python 연계 분석**을 활용해  
수요·접근성·불균형·위험회피 요소를 정량화하고
최종적으로 **DBSCAN + P-median 모델**을 통해 후보지를 도출했습니다.

---

## 분석 목적

- 광진구 내 **아동 친화적 공공 공간 확보 필요성** 확인  
- 주요 생활권 내 **어린이공원 수요 지역 탐색**  
- **접근성·안전·불균형 요소**를 고려한 종합 점수 산출  
- 공간적 군집 기반 **우선 설치 후보지** 도출

## 종합 점수 산출

각 지표를 0~1로 정규화한 뒤, 아래 공식으로 최종 점수를 계산했습니다.

최종 점수
= (5 × 수요)
  (3 × (1 - 불균형))
  (3 × 접근성)
  (위험회피)


**가중치 설정 이유**
- **수요(5)**: 어린이공원 설치 목적과 직접적으로 연결되므로 비중 최대  
- **불균형(3)**: 기존 아동시설의 부족·편중을 보정  
- **접근성(3)**: 실제 이용 가능성과 직결  
- **위험회피(1)**: 안전 취약 지역 개선 필요성 반영  

---

## 입지 가능 지역 1차 도출

다음 조건에 해당하는 격자는 **입지 부적합 지역**으로 제외했습니다.

- 기존 어린이공원에서 **250m 이내**  
- 유흥·단란주점 등 특정 시설에서 **200m 이내**  
- 격자 중심이 **행정구역 외부**에 위치한 경우  

---

## DBSCAN을 활용한 입지 후보지 군집화

**DBSCAN 파라미터**
- eps = 150
- min_samples = 3


종합 점수 **상위 10%** 이상의 격자를 대상으로 군집화 수행  

---

## P-median 기반 최적 위치 선정

DBSCAN으로 도출된 각 클러스터 내부에서  
**P-median 모델**을 활용해 최적 대표 위치를 선정했습니다.

> 목적: “설치 후 이용자들의 이동 거리를 최소화하는 위치 도출”

선정 과정:
- 클러스터 내 격자데이터가 충분한 경우 P-median 모델을 통해 최종 후보지 선정
- 불충분한 경우 기존 점수 기준 사용

최종 후보지 어린이공원이 들어설 수 없는 후보지는 제외(도로,학교 내,... 등)
